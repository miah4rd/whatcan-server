Как это выглядит в итоге
Есть два варианта доставки одного и того же AI-движка. Логика и UI — те же, что ты видишь сейчас в превью (маленькое окошко справа снизу). Меняется только "обёртка":

Вариант 1: Chrome-расширение (production)
Брокер один раз ставит расширение в браузер. Дальше:

Он работает в своей обычной CRM (HubSpot / Pipedrive / Bitrix / свой интерфейс — без разницы)
Поверх CRM в углу экрана висит то самое маленькое окошко-напоминалка
Расширение читает, какой лид открыт в CRM, тянет историю переписки → AI генерит сообщение → брокер жмёт Approve & copy → вставляет в поле ответа CRM
Это не вторая CRM. Это плавающий бейдж поверх существующей. Можно свернуть в иконку.
Вариант 2: Web-демо (то, что сейчас в превью)
То же самое окошко, но без реальной CRM на фоне (фон — декоративная заглушка "типа ваша CRM"). Нужно чтобы:

Показать клиенту/инвестору, как это работает, без установки расширения

Вариант 3: Push на телефон (опционально)
Если брокер не за компьютером — те же напоминания падают как push-уведомление: "Лиду Ивану пора фоллоуп, тапни чтобы увидеть текст".

Правильный порядок установки
НЕ открывай файлы напрямую. Просто распакуй zip — получится папка copilot-extension.
Открой новую вкладку → введи в адресной строке: chrome://extensions
Включи переключатель Developer mode (правый верхний угол)
Нажми кнопку Load unpacked → выбери распакованную папку copilot-extension
Расширение появится в списке. После установки автоматически откроется страница настроек — там playbook будет уже заполнен.
Теперь открой любой сайт (например, свою CRM или просто google.com) — в правом нижнем углу появится плавающее окошко с подсказкой для лида Amelia Reyes.
Если что-то не так
Кнопка Load unpacked серая → не включён Developer mode
Расширение установилось, но окошка нет → проверь что страница не chrome://... (на служебных страницах расширения не работают). Открой обычный сайт.
Окошко появилось, но висит "Preparing…" / ошибка API → проверь интернет; AI зовётся на наш хостед endpoint.
Иконка расширения в тулбаре → клик скрывает/показывает окошко. Правый клик → Options — открывает настройки.
